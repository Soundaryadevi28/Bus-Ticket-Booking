# Bus Ticket Booking System

A professional Python CLI application that automates fare calculation and ticketing for transit stations.

### Key Features
* **Station Mapping:** Uses Python dictionaries to calculate distance-based fares.
* **Smart Validation:** Handles case-sensitivity and whitespace using `.strip()` and `.title()`.
* **Discount Logic:** Automated 20% fare reduction for passengers under age 8.
* **Clean Output:** Generates a structured digital receipt for the user.

### Technical Stack
* **Language:** Python 3
* **Logic:** Data structures, Conditional branching, Input sanitization.
